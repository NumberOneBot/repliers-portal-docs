# Portal Product Guide

**Version:** 1.0
**Last Updated:** February 2025

## What is Portal?

Portal is a **white-label real estate platform** built on the **Repliers API**. It enables brokerages, teams, and agents to launch branded property search websites without building from scratch. Each client gets unique branding, features, and MLS dataвҖ”all powered by a single codebase. The Repliers API supplies the data, a proxy backend sits in front of it (the frontend never calls Repliers directly), and Portal delivers the frontend.

## Who It Serves

Portal is a **B2B** product for real-estate businesses that want their own branded
property-search site вҖ” large brokerages, boutique agencies, teams, and solo agents. Each becomes
a **tenant**: one codebase, its own brand, feature set, and MLS data.

Those businesses serve **end users (B2C)**:

- **Buyers** - Search properties, save favorites, contact agents
- **Sellers** - Get home estimates, find listing agents
- **Renters** - Search rental listings
